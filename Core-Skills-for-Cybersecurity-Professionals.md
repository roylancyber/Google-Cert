# Core Skills for Cybersecurity Professionals

## Transferable Skills

Transferable skills are skills that can be applied across many careers and industries.

Examples:

* Problem-solving
* Analysis
* Written and verbal communication
* Collaboration

## Technical Skills

Technical skills require specialized knowledge, tools, or training.

Examples:

* Programming
* Data analysis
* Computer forensics
* Security Information and Event Management (SIEM) tools

## Digital Forensics

Digital forensic investigators identify, analyze, and preserve digital evidence from computers, networks, and electronic devices.

## Personally Identifiable Information (PII)

PII is information that can be used to identify an individual.

Examples:

* Name
* Date of birth
* Email address
* Phone number

## Sensitive Personally Identifiable Information (SPII)

SPII requires additional protection because it can cause significant harm if exposed.

Examples:

* Bank account numbers
* Medical records
* Social Security numbers
* Credit card information

## SIEM Tools

SIEM stands for Security Information and Event Management.

SIEM tools help security professionals:

* Collect logs
* Monitor security events
* Detect threats
* Analyze risks and vulnerabilities

## What I Learned

Cybersecurity professionals need both technical and transferable skills. Technical skills help identify and respond to threats, while communication, analysis, and problem-solving help security teams work effectively with the rest of the organization.
